Referência rápida de todos os elementos do DSL.

## Estrutura

### Views

| Elemento | Sintaxe | Descrição |
|----------|---------|------------|
| Screen | `screen Nome:` | Tela principal |
| Modal | `modal Nome:` | Dialog centralizado |
| Drawer | `drawer Nome:` | Menu lateral |
| Component | `component Nome:` | Bloco reutilizável |

---

## Tipografia

| Sintaxe | Elemento |
|---------|----------|
| `#` até `######` | Títulos H1-H6 |
| `>` | Parágrafo |
| `>>` | Texto secundário |
| `>>>` | Texto sutil |
| `*>` | Nota/destaque |
| `">` | Citação |

<DslExample title="Tipografia" heightClassName="h-56">

```dsl
screen Example:
  # Título Principal
  ## Seção
  > Parágrafo principal
  >> Texto secundário
  >>> Metadados
  *> Nota importante
  "> Citação
```

</DslExample>

## Elementos Interativos

### Botões

**Tamanhos:** `@` (grande), `@@` (médio), `@@@` (pequeno)

**Variantes:**
- Padrão: `@[Texto](ação)`
- Ghost: `@_[Texto](ação)`
- Outline: `@+[Texto](ação)`
- Secondary: `@-[Texto](ação)`
- Destructive: `@=[Texto](ação)`
- Warning: `@![Texto](ação)`

**Com ícone:** `@[i-NomeIcone Texto](ação)``

### Links e Imagens

- Link: `#[Texto](destino)`
- Imagem: `![Alt](url)`

<DslExample title="Interativos" heightClassName="h-48">

```dsl
screen Actions:
  @[Primário](action)
  @ghost[Ghost](action)
  @outline[Outline](action)
  @destructive[i-Trash Deletar](delete)

  #[Link Externo](https://exemplo.com)
  ![Logo](https://placehold.co/200x60)
```

</DslExample>

## Layout

| Elemento | Sintaxe | Uso |
|----------|---------|-----|
| Container | `container:` | Centraliza conteúdo |
| Row | `row:` | Layout horizontal |
| Col | `col:` | Layout vertical |
| Grid | `grid:` | Layout em grade |
| Card | `card:` | Container com borda |
| Header | `header:` | Cabeçalho fixo |

**Modificadores:** Use `-` para customizar
- Tamanho: `w50`, `h100`, `wfull`
- Espaçamento: `p4`, `p8`, `m4`, `gap4`
- Alinhamento: `center`, `between`, `around`
- Grid: `cols2`, `cols3`, `cols4`

<DslExample title="Layout" heightClassName="h-48">

```dsl
screen Actions:
  container:
    row-between:
      row-start:
        > link 1
        > link 2
      stack:
        > link 3

    row-center:
      card:
        ## Card Centralizado
```

</DslExample>

## Estruturas

### Lista

**Simples:** `list:` + itens com `-`

```dsl
list:
  - Item 1
  - Item 2
```

### Outros

- Separador: `---`
- Navigator: `navigator:` + itens com formato especial
- FAB: `fab[i-NomeIcone](ação)`

<DslExample title="Estruturas" heightClassName="h-56">

```dsl
screen App:
  container-p8:
    list:
      - Tarefa 1
      - Tarefa 2
    
    ---
    
    > Separado por linha
  
  navigator:
    - i-Home Início | Home
    - i-User Perfil | Profile
  
  fab[i-Plus](NewItem)
```

</DslExample>

## Formulários

| Elemento | Sintaxe |
|----------|----------|
| Input | `___:Label{Placeholder}` |
| Senha | `___*:Label{Placeholder}` |
| Desabilitado | `___-:Label{Placeholder}` |
| Select | `___:Label{Placeholder}[Op1 \| Op2]` |
| Checkbox marcado | `[X]` |
| Checkbox desmarcado | `[ ]` |
| Radio selecionado | `(X)` |
| Radio não selecionado | `( )` |

<DslExample title="Formulários" heightClassName="h-56">

```dsl
___:Nome{Digite seu nome}
___*:Senha{Senha secreta}
___:País{Selecione}[Brasil | EUA | Outro]

[X] Aceito termos
[ ] Newsletter

(X) Opção A
( ) Opção B
```

</DslExample>

## Componentes

**Declarar:**
```dsl
component Nome:
  > Conteúdo com %prop
```

**Usar:**
```dsl
$Nome
```

**Com props:**
```dsl
$Nome:
  - valor1 | valor2
```

<DslExample title="Componentes" heightClassName="h-48">

```dsl
component UserCard:
  card:
    ## %name
    >> %email

screen Users:
  $UserCard:
    - João | joao@email.com
  $UserCard:
    - Maria | maria@email.com
```

</DslExample>
