# Themes

## O que é
Abordagem para personalizar o tema global da aplicação ajustando tokens fornecidos pelo sistema (`--primary`, `--muted`, `--radius`).

## Capacidades
- Sobrescrever cores padrão definidas pela engine para adaptar à identidade visual.
- Ajustar tipografia (contrastando `--foreground`, `--muted`), espaçamentos e radius.
- Combinar com `styles:` para tokens customizados e usar em componentes.

## Sintaxe
- `styles:`
- `--primary: oklch(0.68 0.15 210);`
- `--primary-foreground: oklch(0.98 0 0);`
- `--muted: oklch(0.22 0.02 240);`
- `--card: oklch(0.16 0.03 245);`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `--primary`, `--primary-foreground` | Cores primárias | Ajustam botões e CTAs principais. |
| `--secondary`, `--muted` | Cores auxiliares | Influenciam cards, notas, planos de fundo. |
| `--radius` | Borda global | Afeta cards, botões e inputs. |
| Tokens custom | Extensão | Podem ser usados em componentes com CSS inline. |

## Casos de uso
- Aplicar paleta da marca aos botões e títulos.
- Criar temas escuros ou de alto contraste ajustando `--background`, `--foreground`.
- Reaproveitar o mesmo bloco de estilos em várias telas.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
styles:
  --primary: oklch(0.7 0.15 220);
  --primary-foreground: oklch(0.98 0 0);
  --muted: oklch(0.22 0.02 250);
  --card: oklch(0.16 0.03 245);

screen Pricing:
  container:
    # Escolha seu plano
    grid-3:
      card:
        ## Starter
        @[Começar grátis](Signup)
      card-feature:
        ## Business
        @[Assinar Business](upgrade)
      card:
        ## Enterprise
        @[Falar com vendas](contact)
```
</DslPreview>
