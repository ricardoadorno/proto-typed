# Drawer Detail

## O que é
Padrão cookbook que mostra como exibir detalhes de um item dentro de um `drawer`, mantendo a lista principal visível.

## Capacidades
- Abrir `drawer CustomerDetail` a partir de `list` com componentes.
- Passar props para o drawer usando componentes (`component CustomerRow`).
- Combinar com ações (`@ghost`, `@success`) para manipular dados diretamente dentro do painel.

## Sintaxe
- `drawer CustomerDetail:`
- `container:`
- `# %name`
- `>> %company`
- `>>> Último contato: %lastContact`
- `screen Accounts:`
- `list $CustomerRow:`
- `- Ana Souza | Proto Labs | Ontem`
- `component CustomerRow:`
- `card-compact:`
- `row-between:`
- `stack:`
- `## %name`
- `>>> %company`
- `@ghost[i-ChevronRight Ver detalhes](CustomerDetail)`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `drawer CustomerDetail` | Painel de detalhes | Recebe dados via props. |
| `component CustomerRow` | Item da lista | Define o layout de cada linha. |
| `list $CustomerRow` | Coleção de registros | Cada linha passa valores por `|`. |
| Ações internas | Botões do drawer | Permitem agendar follow-up ou fechar. |

## Casos de uso
- Visualizar dados de clientes sem sair da lista principal.
- Adicionar ações rápidas (agendar follow-up, atualizar status) no próprio drawer.
- Reaproveitar o mesmo componente de linha em múltiplas telas.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
component InfoRow:
  row-between:
    >> %label
    >>> %value

drawer CustomerDetail:
  container:
    # %name
    >> %company
    >>> Último contato: %lastContact

    list $InfoRow:
      - Status | Em negociação
      - Valor estimado | R$ 180K
      - Próximo passo | Apresentar proposta final

    card:
      ## Observações
      > Atualize o CRM após a reunião com o cliente.

    @[Marcar follow-up](scheduleFollowUp)
    @ghost[Fechar](close)

component CustomerRow:
  card-compact:
    row-between:
      stack:
        ## %name
        >>> %company
        >>> Último contato: %lastContact
      @ghost[i-ChevronRight Ver detalhes](CustomerDetail)

screen Accounts:
  container:
    # Contas estratégicas
    list $CustomerRow:
      - Ana Souza | Proto Labs | Ontem
      - Bruno Lima | NextWave | 3 dias atrás
      - Carla Dias | FutureSoft | 7 dias atrás
```
</DslPreview>
