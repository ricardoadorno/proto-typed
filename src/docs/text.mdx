# Text

## O que é
Elemento de tipografia responsável por títulos, parágrafos, notas e citações dentro das telas Proto-Typed.

## Capacidades
- Declarar hierarquia de títulos com `#` a `######` para estruturar seções da interface.
- Produzir parágrafos de diferentes pesos: `>`, `>>`, `>>>`.
- Destacar informações com notas (`*>`) e registrar depoimentos com citações (`">`).
- Combinar conteúdo textual com layout (`container`, `stack`, `card`) e elementos de ação como `button`.
- Inserir ícones embutidos (ex.: `i-Check`) ao lado de textos usando a sintaxe padrão da DSL.

## Sintaxe
- `# Heading principal`
- `## Subtítulo`
- `> Parágrafo principal`
- `>> Texto de apoio`
- `>>> Texto sutil ou metadado`
- `*> Nota importante`
- `"> Citação com autoria`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `# … ######` | Títulos (H1–H6) | Use apenas um `#` por tela para o título principal. |
| `>` | Parágrafo principal | Ideal para copy primária ou introduções. |
| `>>` | Texto secundário | Complementa informações principais. |
| `>>>` | Texto sutil | Metadados, timestamps ou labels discretos. |
| `*>` | Nota/Callout | Destaca alertas, dicas e avisos. |
| `">` | Citação | Exibe depoimentos ou referências. |

## Casos de uso
- `> Overview: Utilize parágrafos principais para contextualizar dashboards e relatórios.`
- `*> Nota: Destaque checklists ou avisos críticos dentro de um card.`
- `"> Citação: Traga feedback de usuários ou mensagens de confirmação.`
- `>>> Metadado: Registre data/hora de última atualização de um bloco.`

    >>> Atualizado em 18/09 às 09:30

    card:
      *> Próxima revisão em 7 dias.
