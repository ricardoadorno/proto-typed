# Login Screen

## O que é
Guia cookbook que monta uma tela de autenticação completa com inputs, botões de ação e links auxiliares.

## Capacidades
- Organizar formulário em `container` e `card` com `stack` interna.
- Validar campos essenciais (`___email`, `___password`) e ativar ações (`@[Entrar]`).
- Fornecer fluxos paralelos (recuperar senha, criar conta) com botões e links.
- Reutilizar a mesma tela em conjunto com modais ou dashboards.

## Sintaxe
- `screen Login:`
- `container:`
- `card:`
- `___email: Email{seu@dominio.com}`
- `___password: Senha{••••}`
- `[X] Lembrar de mim`
- `row-end:`
- `@ghost[Esqueci a senha](Recover)`
- `@[Entrar](Dashboard)`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `screen Login` | Tela principal | Exibe formulário e CTAs. |
| Inputs (`___email`, `___password`) | Captura dados | Podem receber `required` nos atributos. |
| Links auxiliares | Navegação | Use `@ghost` ou `#[...]` para rotas secundárias. |
| CTA principal | Botão `@` | Direciona para dashboard ou próxima etapa. |

## Casos de uso
- Tela inicial de acesso para apps internos ou SaaS.
- Workflow com recuperação de senha via outra `screen`. 
- Encadeamento com `modal` de autenticação em dois fatores.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
screen Login:
  container:
    # Acesse sua conta
    card:
      stack:
        ___email: E-mail{usuario@empresa.com} | required
        ___password: Senha{Mínimo 8 caracteres} | required minlength="8"
        [X] Permanecer conectado

      row-end:
        @ghost[Esqueci a senha](Recover)
        @[Entrar](Dashboard)

screen Recover:
  container-narrow:
    # Recuperar senha
    ___email: E-mail cadastrado{usuario@empresa.com} | required
    row-end:
      @ghost[Cancelar](-1)
      @[Enviar instruções](sendReset)

screen Signup:
  container-narrow:
    # Criar conta
    stack:
      ___: Nome completo{Seu nome}
      ___email: E-mail{email@empresa.com}
      ___password: Senha{••••••••} | minlength="8"
    row-end:
      @ghost[Voltar](Login)
      @primary[Começar agora](createAccount)
```
</DslPreview>
