# Grid

## O que é
Layout em grade responsável por distribuir elementos em colunas, garantindo alinhamento consistente e responsividade.

## Capacidades
- Declarar grades fixas (`grid-2`, `grid-3`, `grid-4`) e automáticas (`grid-auto`).
- Acomodar qualquer elemento (cards, componentes customizados, listas) mantendo espaçamento uniforme.
- Combinar com `container` ou `card` para criar dashboards e catálogos.
- Adaptar-se a diferentes larguras sem exigir modificadores adicionais.

## Sintaxe
- `grid-2:`
- `card:`
- `# Item A`
- `card:`
- `# Item B`
- `grid-auto:`
- `card:`
- `# Item responsivo`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `grid-2` | 2 colunas fixas | Ideal para painéis e listagens curtas. |
| `grid-3` | 3 colunas | Padrão para dashboards e métricas. |
| `grid-4` | 4 colunas | Use quando houver espaço amplo/desktops. |
| `grid-auto` | Auto-fit | Ajusta número de colunas conforme largura disponível. |
| Conteúdo interno | Slot | Normalmente `card`, `component`, `list`, etc. |

## Casos de uso
- `grid-3:` Exibir métricas principais com cards individuais.
- `grid-2:` Comparar planos, features ou cards com detalhes.
- `grid-auto:` Listar coleções de itens que devem “quebrar” responsivamente.
- `grid-4:` Mostrar thumbnails ou indicadores em telas largas.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
screen Analytics:
  container-wide:
    # Indicadores principais

    grid-3:
      card:
        ## Usuários ativos
        # 12.304
      card:
        ## Conversão
        # 23.5%
      card:
        ## NPS
        # 72

    grid-2:
      card:
        ## Últimas releases
        list:
          - v1.4.0 | 12/09
          - v1.3.2 | 28/08
      card:
        ## Próximos marcos
        list:
          - Pesquisa usuários | Outubro
          - Revisão monetização | Novembro
```
</DslPreview>

<DslPreview title="Exemplo 2">
```dsl
component FeatureCard:
  card:
    ## %title
    > %description
    @[Saiba mais](%destination)

screen ProductHighlights:
  container:
    # Destaques do produto
    grid-auto:
      $FeatureCard:
        - Lançamento rápido | Monte protótipos em minutos | QuickStart
      $FeatureCard:
        - Time alinhado | Documentação sempre sincronizada | Guides
      $FeatureCard:
        - Exportar facilmente | Gere relatórios com um clique | Reports

  container:
    row-end:
      @outline[Ver roadmap](Roadmap)
      @[Começar agora](Signup)
```
</DslPreview>
