# Sistema de Layout

import { DslExample } from "../../components/docs";


Organize elementos com **layouts canônicos pré-configurados**. Cada layout representa um preset sem modificadores dinâmicos.

**Importante:** Não há mais modificadores inline. Use apenas layouts semânticos como `row-center`, `stack-tight`, `container-narrow`, `grid-3`, etc.

---

## Container

Centraliza conteúdo com max-width apropriada.

**Variações:**
- `container:` → Container padrão
- `container-narrow:` → Container estreito (sm)
- `container-wide:` → Container largo (lg)
- `container-full:` → Largura total

<DslExample title="Containers" description="Wrappers centralizados">

```dsl
screen ContainerExample:
  container:
    # Container Padrão
    > Conteúdo centralizado com max-width.
  
  container-narrow:
    # Container Estreito
    > Ideal para texto e formulários.
  
  container-wide:
    # Container Largo
    > Para dashboards e conteúdo rico.
```

</DslExample>

---

## Stack (Vertical)

Organiza elementos verticalmente com gap.

**Variações:**
- `stack:` → Stack padrão (gap-4)
- `stack-tight:` → Stack com gap pequeno (gap-2)
- `stack-loose:` → Stack com gap grande (gap-8)
- `stack-flush:` → Sem gap

<DslExample title="Stacks" description="Layout vertical">

```dsl
screen StackExample:
  container:
    # Stack Padrão
    stack:
      @[Botão 1](action)
      @[Botão 2](action)
      @[Botão 3](action)
    
    # Stack Tight
    stack-tight:
      > Item 1
      > Item 2
      > Item 3
```

</DslExample>

---

## Row (Horizontal)

Organiza elementos horizontalmente.

**Variações:**
- `row-start:` → Alinhado ao início
- `row-center:` → Centralizado
- `row-between:` → Espaçamento entre elementos
- `row-end:` → Alinhado ao fim

<DslExample title="Rows" description="Layout horizontal">

```dsl
screen RowExample:
  container:
    row-center:
      @[Botão 1](action)
      @[Botão 2](action)
      @[Botão 3](action)
    
    row-between:
      @[Esquerda](left)
      @[Direita](right)
```

</DslExample>

---

## Grid

Layout em grade com colunas predefinidas.

**Variações:**
- `grid-2:` → 2 colunas
- `grid-3:` → 3 colunas
- `grid-4:` → 4 colunas
- `grid-auto:` → Auto-fit responsivo

<DslExample title="Grids" description="Layout em grade">

```dsl
screen GridExample:
  container:
    grid-3:
      card:
        ## Item 1
      card:
        ## Item 2
      card:
        ## Item 3
```

</DslExample>

---

## Card

Container com borda, sombra e padding.

**Variações:**
- `card:` → Card padrão
- `card-compact:` → Card com menos padding
- `card-feature:` → Card destacado

<DslExample title="Cards" description="Containers destacados">

```dsl
screen CardExample:
  container:
    grid-2:
      card:
        ## Card Padrão
        > Padding normal
      
      card-compact:
        ## Card Compacto
        > Menos padding
```

</DslExample>

## Header

Cabeçalho fixo no topo.

<DslExample title="Header" description="Cabeçalho de página">

```dsl
screen App:
  header:
    # Minha App
    @ghost[Menu](menu)
  
  container:
    # Conteúdo Principal
    > Lorem ipsum dolor sit amet
```

</DslExample>

---

## Sidebar

Layout de barra lateral.

<DslExample title="Sidebar" description="Navegação lateral">

```dsl
screen AppLayout:
  sidebar:
    # Menu
    list:
      - Home | Home
      - Settings | Settings
  
  container:
    # Conteúdo
    > Área principal
```

</DslExample>

## Casos de Uso

### Dashboard Layout

<DslExample title="Dashboard" description="Grid + cards">

```dsl
screen Dashboard:
  container:
    # Dashboard
    
    grid-3:
      card:
        ## Usuários
        >>> 1,234
      
      card:
        ## Vendas
        >>> R$ 45,678
      
      card:
        ## Conversão
        >>> 23.5%
```

</DslExample>

### Header com Ações

<DslExample title="Header" description="Cabeçalho com navegação">

```dsl
screen AppHeader:
  header:
    # Minha App
    row-end:
      @ghost[Busca](search)
      @ghost[Perfil](profile)
```

</DslExample>

### Form Layout

<DslExample title="Formulário em Grid" description="2 colunas de inputs">

```dsl
screen FormGrid:
  container:
    grid-2:
      ___: Nome{Nome}
      ___email: Email{Email}
      
      ___: Telefone{Telefone}
      ___: CEP{CEP}
    
    @[Enviar](submit)
```

</DslExample>

## Layouts Disponíveis

| Layout | Variações | Descrição |
|--------|-----------|-----------|
| **Container** | `container`, `container-narrow`, `container-wide`, `container-full` | Wrappers centralizados |
| **Stack** | `stack`, `stack-tight`, `stack-loose`, `stack-flush` | Vertical com gap |
| **Row** | `row-start`, `row-center`, `row-between`, `row-end` | Horizontal |
| **Grid** | `grid-2`, `grid-3`, `grid-4`, `grid-auto` | Grade de colunas |
| **Card** | `card`, `card-compact`, `card-feature` | Containers destacados |
| **Header** | `header` | Cabeçalho fixo |
| **Sidebar** | `sidebar` | Navegação lateral |
| **List** | `list` | Lista de itens |
| **Navigator** | `navigator` | Navegação inferior |
| **Fab** | `fab` | Botão flutuante |
| **Separator** | `---` | Linha divisória |


> **Atenção:** Modificadores dinâmicos como `row-w50-center-p4` **não são mais suportados**. Use apenas os layouts canônicos listados acima.
