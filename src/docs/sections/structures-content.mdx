# Estruturas de Conteúdo

import { DslExample } from "../../components/docs";

Elementos estruturais para organizar conteúdo: listas, cards, headers, navegadores, FABs e separadores.

---

## List (Lista)

**Sintaxe:** `list:` seguido de itens com `-`

<DslExample title="Lista Simples" description="Itens com marcadores">

```dsl
screen TodoList:
  container-p8:
    # Tarefas
    
    list:
      - Fazer compras
      - Limpar casa
      - Estudar DSL
```

</DslExample>

### Lista com Conte\u00fado Complexo

<DslExample title="Lista Complexa" description="Itens com m\u00faltiplos elementos">

```dsl
screen Features:
  container-p8:
    # Funcionalidades
    
    list:
      - ## Prototipagem R\u00e1pida
        > Crie interfaces em minutos
        @@[Saiba Mais](docs)
      
      - ## Sistema de Temas
        > 12 temas dispon\u00edveis
        @@[Ver Temas](themes)
```

</DslExample>

## Card

**Sintaxe:** `card:` ou `card-{modificadores}:`

Container com borda e sombra para agrupar conte\u00fado.

<DslExample title="Cards" description="Containers destacados">

```dsl
screen Dashboard:
  container-p8:
    grid-cols2-gap4:
      card-p6:
        ## Usu\u00e1rios Ativos
        # 1,234
        >>> +12% este m\u00eas
      
      card-p6:
        ## Receita Total
        # R$ 45,678
        >>> +8% este m\u00eas
```

</DslExample>

## Header

**Sintaxe:** `header:` ou `header-modifiers:`

Cabeçalho fixo no topo da tela.

<DslExample title="Header" description="Cabeçalho com ações">

```dsl
screen App:
  header-between-p4:
    # Minha App
    row-gap2:
      @@@_[i-Search Busca](search)
      @@@_[i-User Perfil](profile)
  
  container-p8:
    # Conteúdo Principal
    > Lorem ipsum dolor sit amet
```

</DslExample>

---

## Navigator

**Sintaxe:** `navigator:` seguido de itens com formato `- texto | destino`

Barra de navegação inferior fixa.

<DslExample title="Navigator" description="Navegação inferior">

```dsl
screen Home:
  container:
    # Bem-vindo
    > Conteúdo da tela
  
  navigator:
    - Home | Home
    - Search | Search
    - Profile | Profile
    - Settings | Settings
```

</DslExample>

---

## FAB (Floating Action Button)

**Sintaxe:** `fab:` seguido de item `- ícone | ação`

Botão flutuante fixo no canto inferior direito.

<DslExample title="FAB" description="Botão de ação primária">

```dsl
screen Messages:
  container:
    # Mensagens
    
    list:
      - Conversa com João
      - Conversa com Maria
      - Conversa com Pedro
  
  fab:
    - + | NewMessage
```

</DslExample>

---

## Separator (Separador)

**Sintaxe:** `---`

Linha horizontal para dividir se\u00e7\u00f5es.

<DslExample title="Separador" description="Divis\u00f3rias visuais">

```dsl
screen Profile:
  container-p8:
    # Jo\u00e3o Silva
    >> Desenvolvedor Full Stack
    
    ---
    
    ## Contato
    > Email: joao@email.com
    > Telefone: (11) 99999-9999
    
    ---
    
    ## Biografia
    > Apaixonado por tecnologia e c\u00f3digo limpo.
```

</DslExample>

## Casos de Uso

### App Mobile Completo

<DslExample title="App com Estruturas" description="Header + Navigator + FAB">

```dsl
screen TaskApp:
  header-between-p4:
    # Tarefas
    @@@_[i-Filter Filtro](filter)
  
  container-p8:
    list:
      - [X] Fazer compras
      - [ ] Limpar casa
      - [ ] Estudar DSL
  
  navigator:
    - i-Home Hoje | TaskApp
    - i-Calendar Semana | Week
    - i-CheckSquare Concluídas | Done
  
  fab[i-Plus](NewTask)
```

</DslExample>

## Combina\u00e7\u00f5es

Todas as estruturas suportam modificadores inline:

```dsl
card-p8-w75-center:
  # Card Customizado

header-between-center-p6:
  # Header com Padding e Alinhamento

list:
  - Item com card:
    card-p4-m2:
      > Conte\u00fado aninhado
```
