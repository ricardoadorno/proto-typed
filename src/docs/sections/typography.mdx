# Tipografia

import { DslExample } from "../../components/docs";

Elementos semânticos para estruturar texto.

---

## Títulos

**Sintaxe:** `#` até `######` (H1 a H6)

<DslExample title="Títulos" heightClassName="h-56">

```dsl
# Título 1
## Título 2
### Título 3
#### Título 4
##### Título 5
###### Título 6
```

</DslExample>

---

## Textos

| Sintaxe | Elemento | Uso |
|---------|----------|-----|
| `>` | Parágrafo | Texto principal |
| `>>` | Texto | Texto secundário |
| `>>>` | Texto sutil | Metadados, timestamps |

<DslExample title="Variações de Texto" heightClassName="h-40">

```dsl
> Parágrafo principal para conteúdo importante
>> Texto secundário para informações complementares
>>> Texto sutil para metadados ou detalhes discretos
```

</DslExample>

---

## Elementos Especiais

| Sintaxe | Elemento | Uso |
|---------|----------|-----|
| `*>` | Nota | Destaques, avisos, dicas |
| `">` | Citação | Citações, depoimentos |

<DslExample title="Notas e Citações" heightClassName="h-40">

```dsl
*> Nota importante: Esta informação merece destaque visual

"> A simplicidade é a máxima sofisticação. — Leonardo da Vinci
```

</DslExample>
