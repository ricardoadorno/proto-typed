Use semantic tokens for headings, body, notes, and quotes.

## Headings
<DslExample title="Headings" heightClassName="h-56">

```dsl
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

</DslExample>

## Text Variants
<DslExample title="Text variants" heightClassName="h-40">

```dsl
> Regular body text for paragraphs
*> Note text for hints or callouts
" > Quote text for citations
```

</DslExample>

Guidelines:
- Keep headings short and scannable.
- Combine with `card` and layout primitives for rhythm.
