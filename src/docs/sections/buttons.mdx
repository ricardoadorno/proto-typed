# Botões

import { DslExample } from "../../components/docs";

**Sintaxe:** `@[Texto](ação)`

**Com variante e tamanho:** `@<variante>-<tamanho>[Texto](ação)`

---

## Variantes

**Sintaxe:** `@<variante>[Texto]` (opcional, padrão é primary)

| Variante | Sintaxe | Uso |
|----------|---------|-----|
| Primary | `@primary` ou `@` | Ação primária (padrão) |
| Secondary | `@secondary` | Ação secundária |
| Outline | `@outline` | Ação com borda |
| Ghost | `@ghost` | Ação discreta |
| Destructive | `@destructive` | Deletar/remover |
| Link | `@link` | Estilo de link |
| Success | `@success` | Confirmação positiva |
| Warning | `@warning` | Atenção/alerta |

<DslExample title="Variantes" description="8 estilos visuais">

```dsl
screen ButtonVariants:
  container:
    stack:
      @[Primary](action)
      @secondary[Secondary](action)
      @outline[Outline](action)
      @ghost[Ghost](action)
      @destructive[Destructive](delete)
      @link[Link](action)
      @success[Success](action)
      @warning[Warning](warning)
```

</DslExample>

## Tamanhos

**Sintaxe:** `-<tamanho>` após a variante (opcional, padrão é md)

| Tamanho | Sintaxe |
|---------|---------|
| Extra Small | `-xs` |
| Small | `-sm` |
| Medium | `-md` (padrão) |
| Large | `-lg` |

<DslExample title="Tamanhos" description="4 tamanhos disponíveis">

```dsl
screen ButtonSizes:
  container:
    stack:
      @-xs[Extra Small](action)
      @-sm[Small](action)
      @-md[Medium](action)
      @-lg[Large](action)
```

</DslExample>

---

## Combinando Variante e Tamanho

<DslExample title="Variantes + Tamanhos" description="Combinações possíveis">

```dsl
screen ButtonCombinations:
  container:
    stack:
      @secondary-lg[Secondary Large](action)
      @outline-sm[Outline Small](action)
      @ghost-xs[Ghost Extra Small](action)
      @destructive-lg[Delete Large](delete)
```

</DslExample>

## Ações

| Tipo | Sintaxe | Exemplo |
|------|---------|----------|
| Navegação interna | `(NomeDaTela)` | `@[Dashboard](Dashboard)` |
| URL externa | `(https://...)` | `@[Site](https://exemplo.com)` |
| JavaScript | `(código)` | `@[Alert](alert('Oi!'))` |
| Voltar | `(-1)` | `@[Voltar](-1)` |

<DslExample title="Tipos de Ação" description="Navegação e JavaScript">

```dsl
screen Home:
  container:
    stack:
      @[Ir para Dashboard](Dashboard)
      @[Abrir GitHub](https://github.com)
      @[Mostrar Mensagem](alert('Clicado!'))

screen Dashboard:
  container:
    # Dashboard
    @[Voltar](Home)
```

</DslExample>

## Casos de Uso

### Formulário

<DslExample title="Formulário com Botões" description="CTAs e ações secundárias">

```dsl
screen FormExample:
  card:
    # Cadastro
    ___: Nome{Digite seu nome}
    ___email: Email{email@exemplo.com}
    
    row-between:
      @[Cadastrar](register)
      @outline[Cancelar](cancel)
```

</DslExample>

### Card de Produto

<DslExample title="Card com Múltiplos Botões" description="Hierarquia de ações">

```dsl
screen ProductCard:
  card:
    ![Produto](https://via.placeholder.com/200)
    ## Produto XYZ
    >> R$ 99,90
    
    row-between:
      @secondary[Detalhes](ProductDetail)
      @[Comprar](buy)
```

</DslExample>

### Ações Destrutivas

<DslExample title="Confirmação de Exclusão" description="Variante destrutiva">

```dsl
screen DeleteConfirm:
  container:
    card:
      # Confirmar Exclusão
      > Esta ação não pode ser desfeita.
      
      row-between:
        @destructive[Excluir Permanentemente](delete)
        @outline[Cancelar](cancel)
```

</DslExample>
