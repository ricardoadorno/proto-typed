## Formulário Multi-Step

<DslExample title="Cadastro em Etapas" heightClassName="h-64">

```dsl
screen Step1:
  container-p8:
    # Passo 1 de 3
    >>> Dados Pessoais
    
    ___:Nome Completo{Seu nome}
    ___:Email{email@exemplo.com}
    ___:Telefone{(00) 00000-0000}
    
    @[i-ArrowRight Próximo: Endereço](Step2)

screen Step2:
  container-p8:
    # Passo 2 de 3
    >>> Endereço
    
    ___:CEP{00000-000}
    ___:Rua{Nome da rua}
    ___:Número{123}
    ___:Estado{Selecione}[SP | RJ | MG]
    
    row-gap4:
      @@+[Voltar](Step1)
      @[i-ArrowRight Próximo: Confirmação](Step3)

screen Step3:
  container-p8:
    # Passo 3 de 3
    >>> Revisão
    
    card-p6:
      ## Seus Dados
      > Nome: João Silva
      > Email: joao@email.com
      > Endereço: Rua ABC, 123 - SP
    
    row-gap4:
      @@+[Voltar](Step2)
      @[i-Check Finalizar](Success)
```

</DslExample>
