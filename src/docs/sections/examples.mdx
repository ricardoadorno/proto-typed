## App de Tarefas

    # Nova Tarefa
    ___: Título{Nome da tarefa}
    ___: Prioridade{Selecione}[Alta | Média | Baixa]

## Formulário Multi-Step

<DslExample title="Cadastro em Etapas" heightClassName="h-64">

```dsl
screen Step1:
  container:
    # Passo 1 de 3
    >>> Dados Pessoais
    
    ___: Nome Completo{Seu nome}
    ___email: Email{email@exemplo.com}
    ___: Telefone{(00) 00000-0000}
    
    @[Próximo: Endereço](Step2)

screen Step2:
  container:
    # Passo 2 de 3
    >>> Endereço
    
    ___: CEP{00000-000}
    ___: Rua{Nome da rua}
    ___number: Número{123}
    ___: Estado{Selecione}[SP | RJ | MG]
    
    row-between:
      @outline[Voltar](Step1)
      @[Próximo: Confirmação](Step3)

screen Step3:
  container:
    # Passo 3 de 3
    >>> Revisão
    
    card:
      ## Seus Dados
      > Nome: João Silva
      > Email: joao@email.com
      > Endereço: Rua ABC, 123 - SP
    
    row-between:
      @outline[Voltar](Step2)
      @success[Finalizar](Success)
```

</DslExample>

## Mensagens/Chat

<DslExample title="App de Mensagens" heightClassName="h-64">

```dsl

  container:
    # Menu
    list:
      - Perfil | Profile
      - Configurações | Settings

      - Maria Santos | há 1 hora | @[Abrir](ChatMaria)
      - Pedro Costa | ontem | @[Abrir](ChatPedro)
  
  fab:
    - + | NewChat
  
  navigator:
    - Chats | Chats
    - Contatos | Contacts
    - Perfil | Profile
```

</DslExample>
