A few practical snippets to accelerate your prototyping flow.

## Dashboard
<DslExample title="Dashboard layout">

```dsl
screen Dashboard:
  row:
    col:
      # Welcome
      > You have 3 tasks today
      @[New Task]()
    col:
      list:
        - [Inbox](Inbox)Email{2 unread}[Open](Inbox)
        - [Tasks](Tasks)Today{3 items}[Open](Tasks)
```

</DslExample>
