## Formulário Multi-Step

<DslExample title="Cadastro em Etapas" heightClassName="h-64">

```dsl
screen Step1:
  container-p8:
    # Passo 1 de 3
    >>> Dados Pessoais
    
    ___:Nome Completo{Seu nome}
    ___:Email{email@exemplo.com}
    ___:Telefone{(00) 00000-0000}
    
    @[Próximo: Endereço]{arrow-right}(Step2)

screen Step2:
  container-p8:
    # Passo 2 de 3
    >>> Endereço
    
    ___:CEP{00000-000}
    ___:Rua{Nome da rua}
    ___:Número{123}
    ___:Estado{Selecione}[SP | RJ | MG]
    
    row-gap4:
      @@+[Voltar](Step1)
      @[Próximo: Confirmação]{arrow-right}(Step3)

screen Step3:
  container-p8:
    # Passo 3 de 3
    >>> Revisão
    
    card-p6:
      ## Seus Dados
      > Nome: João Silva
      > Email: joao@email.com
      > Endereço: Rua ABC, 123 - SP
    
    row-gap4:
      @@+[Voltar](Step2)
      @[Finalizar]{check}(Success)
```

</DslExample>

## Mensagens/Chat

<DslExample title="App de Mensagens" heightClassName="h-64">

```dsl
drawer Menu:
  container-p8:
    # Menu
    list:
      - i-User Perfil | Profile
      - i-Settings Configurações | Settings
      - i-HelpCircle Ajuda | Help
    ---
    @_[Sair]{log-out}(logout)

screen Chats:
  header-between-p4:
    @@@_[Menu]{menu}(Menu)
    # Mensagens
    @@@_[Nova]{plus}(NewChat)
  
  container-p8:
    list:
      - João Silva >>> há 2 min @[Abrir](ChatJoao)
      - Maria Santos >>> há 1 hora @[Abrir](ChatMaria)
      - Pedro Costa >>> ontem @[Abrir](ChatPedro)
  
  fab{plus}(NewChat)
  
  navigator:
    - i-MessageCircle Chats | Chats
    - i-Users Contatos | Contacts
    - i-User Perfil | Profile
```

</DslExample>
