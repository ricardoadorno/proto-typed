# Layout Primitives

import { DslExample } from "../../components/docs";

Compose responsive prototypes with containers, rows, columns, grids, and cards.

## Container
<DslExample title="Container" heightClassName="h-28">

```dsl
container:
  >> Constrains content width and adds spacing
```

</DslExample>

## Rows & Columns
<DslExample title="Rows & columns" heightClassName="h-40">

```dsl
row:
  col:
    > Left
  col:
    > Right
```

</DslExample>

## Grid
<DslExample title="Grid" heightClassName="h-40">

```dsl
grid:
  > Grid item A
  > Grid item B
  > Grid item C
```

</DslExample>

## Cards
<DslExample title="Card" heightClassName="h-40">

```dsl
card:
  # Title
  > Body text inside a card
  @[Action]()
```

</DslExample>

Tips:
- Keep nesting shallow where possible.
- Place actions at the end of groups for scannability.
