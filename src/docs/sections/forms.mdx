# Forms

import { DslExample } from "../../components/docs";

Use compact tokens for inputs, passwords, disabled fields, and selects.

## Text Inputs
<DslExample title="Text input" heightClassName="h-28">

```dsl
___:Label{Placeholder}
```

</DslExample>

## Password Inputs
<DslExample title="Password input" heightClassName="h-28">

```dsl
___*:Password{Enter password}
```

</DslExample>

## Disabled Inputs
<DslExample title="Disabled input" heightClassName="h-28">

```dsl
___-:Read Only{Unavailable}
```

</DslExample>

## Selects
<DslExample title="Select field" heightClassName="h-28">

```dsl
___:Fruit{Choose}[Apple | Banana | Orange]
```

</DslExample>

## Checkboxes & Radio
<DslExample title="Checkboxes & radios" heightClassName="h-40">

```dsl
[ ] Remember me
[X] Subscribed
( ) Free
(X) Pro
```

</DslExample>

Combine form fields with `card`, `row`, and `col` for structure.
