# Ícones

import { DslExample } from "../../components/docs";

Sistema de ícones usando **Lucide Icons** (https://lucide.dev).

---

## Como Funciona

Ícones são adicionados usando o prefixo `i-` seguido do nome do ícone.

**Biblioteca:** [Lucide Icons](https://lucide.dev) - mais de 1000 ícones disponíveis

---

## Onde Usar

| Elemento | Sintaxe | Exemplo |
|----------|---------|---------|
| Botões | `i-Icone` antes do texto | `@[i-Save Salvar](action)` |
| Navigator | `i-NomeIcone` antes do texto | `- i-Home Início \| Home` |
| FAB | `i-Icone` entre colchetes | `fab[i-Plus](NewItem)` |

---

## Botões com Ícones

<DslExample title="Botões + Ícones" heightClassName="h-48">

```dsl
screen IconButtons:
  container-p8:
    row-gap4:
      @[i-Save Salvar](save)
      @[i-Edit Editar](edit)
      @[i-Share Compartilhar](share)
      @[i-Trash2 Deletar](delete)
    
    row-gap4:
      @@+[i-Download Download](download)
      @@@_[i-Settings Configurações](settings)
```

</DslExample>

## Navigator com Ícones

<DslExample title="Navegação com Ícones" heightClassName="h-56">

```dsl

  container-p8:
    # Início
    > Conteúdo da tela
  
  navigator:
    - i-Home Início | Home
    - i-Search Buscar | Search
    - i-User Perfil | Profile
    - i-Settings Config | Settings
```

</DslExample>

## Ícones Populares

| Nome | Ícone | Uso Comum |
|------|-------|-----------|
| `Home` | 🏠 | Tela inicial |
| `Search` | 🔍 | Busca |
| `User` | 👤 | Perfil |
| `Settings` | ⚙️ | Configurações |
| `Plus` | ➕ | Adicionar |
| `Trash2` | 🗑️ | Deletar |
| `Edit` | ✏️ | Editar |
| `Save` | 💾 | Salvar |
| `Share` | 📤 | Compartilhar |
| `Download` | ⬇️ | Download |
| `Mail` | 📧 | Email |
| `Bell` | 🔔 | Notificações |
| `Check` | ✓ | Confirmar |
| `X` | ✕ | Fechar |

**Importante:** Nomes usam **PascalCase** - primeira letra maiúscula.

## Casos de Uso

### Toolbar

<DslExample title="Toolbar com Ações" heightClassName="h-40">

```dsl
screen Editor:
  header-between-p4:
    # Editor
    row-gap2:
      @@@_[i-Save Salvar](save)
      @@@_[i-Undo Desfazer](undo)
      @@@_[i-Redo Refazer](redo)
```

</DslExample>

</DslExample>
