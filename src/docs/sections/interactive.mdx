# Interactive Elements

import { DslExample } from "../../components/docs";

Buttons, links, and images enable flows and interactions.

## Buttons
Variants are prefixed with `@` modifiers:
- Ghost: `@_`
- Outline: `@+`
- Secondary: `@-`
- Destructive: `@=`
- Warning: `@!`

<DslExample title="Button variants" heightClassName="h-48">

```dsl
@[Primary](go)
@_[Ghost](go)
@+[Outline](go)
@-[Secondary](go)
@=[Delete](danger)
@![Warn](warn)
```

</DslExample>
