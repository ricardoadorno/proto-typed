### Variants

Apply a variant before a button to change styling:

<DslExample
	title="Button Variants"
	description="Ghost @_ • Outline @+ • Secondary @- • Destructive @= • Warning @!"
	heightClassName="h-48"
	code={`
@_[Ghost](ghost)
@+[Outline](outline)
@-[Secondary](secondary)
@=[Destructive](delete)
@![Warning](warn)
	`}
/>

Variants compose in lists and other containers.

## Navigation

- Actions like `(Destination)` navigate to a `screen Destination` if present.
- If `Destination` matches a named element (`modal Name`, `drawer Name`), that element is activated.
- Use `(close)` within modals/drawers to dismiss them.
