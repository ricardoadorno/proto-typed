# Navigation

import { DslExample } from "../../components/docs";

Navigation connects screens and toggles named overlays like modals and drawers.

## Screens
<DslExample title="Screen-to-screen navigation">

```dsl
screen Home:
  > Welcome
  @[Go to About](About)

screen About:
  > About screen
  @[Back](Home)
```

</DslExample>

## Links vs Buttons
- `#[Link]()` for inline navigation
- `@[Button]()` for call-to-action

<DslExample title="Links vs buttons" heightClassName="h-40">

```dsl
> Explore our #[examples](/docs/examples)
@[Open Examples](/docs/examples)
```

</DslExample>

## Named Elements
<DslExample title="Declare overlays">

```dsl
modal Help:
  # Need Help?
  > This modal opens from buttons
  @[Close]()

drawer Nav:
  > Drawer content
  #[Home](Home)
```

</DslExample>

Activate by name via actions:
<DslExample title="Activate overlays" heightClassName="h-28">

```dsl
@[Show Help](Help)
@[Open Nav](Nav)
```

</DslExample>

Notes:
- Unreferenced overlays remain hidden by default.
- Actions can target screens, modals, or drawers.
