# Navegação

import { DslExample } from "../../components/docs";

Sistema de navegação entre telas com links e botões.

---

## Screens (Telas)

**Sintaxe:** `screen NomeDaTela:`

Cada tela é uma view separada que pode ser acessada via navegação.

<DslExample title="Navegação entre Telas" heightClassName="h-48">

```dsl
screen Home:
  container:
    # Bem-vindo
    > Tela inicial da aplicação
    @[Ver Perfil](Profile)
    @[Configurações](Settings)

screen Profile:
  container:
    # Perfil
    > Informações do usuário
    @[Voltar](Home)

screen Settings:
  container:
    # Configurações
    @[Voltar](Home)
```

</DslExample>

---

## Tipos de Navegação

| Tipo | Sintaxe | Exemplo |
|------|---------|---------|
| Tela interna | `(NomeDaTela)` | `@[Dashboard](Dashboard)` |
| URL externa | `(https://...)` | `@[Site](https://exemplo.com)` |
| Voltar | `(-1)` | `@[Voltar](-1)` |
| Modal/Drawer | `(NomeOverlay)` | `@[Ajuda](HelpModal)` |

## Links vs Botões

**Links:** `#[texto](destino)` - para navegação inline
**Botões:** `@[texto](destino)` - para call-to-action

<DslExample title="Links vs Botões" heightClassName="h-40">

```dsl
screen Docs:
  container:
    # Documentação
    
    > Leia mais sobre #[componentes](Components) e #[layouts](Layouts)
    
    @[Voltar ao Início](Home)
```

</DslExample>

## Navegação com Navigator

Use `navigator:` para criar barra de navegação inferior (mobile).

<DslExample title="Navigator (Barra Inferior)" heightClassName="h-56">

```dsl
screen Home:
  container:
    # Início
    > Conteúdo principal
  
  navigator:
    - Início | Home
    - Buscar | Search
    - Perfil | Profile
    - Config | Settings

screen Search:
  container:
    # Buscar
  
  navigator:
    - Início | Home
    - Buscar | Search
    - Perfil | Profile
    - Config | Settings
```

</DslExample>

## Fluxos de Navegação

<DslExample title="Fluxo Multi-Tela" heightClassName="h-64">

```dsl
screen Welcome:
  container-p8-center:
    # Bem-vindo
    > Cadastro em 3 passos
    @[Começar](Step1)

screen Step1:
  container-p8:
    # Passo 1: Dados Pessoais
    ___:Nome{Seu nome}
    ___:Email{email@exemplo.com}
    @[Próximo](Step2)

screen Step2:
  container-p8:
    # Passo 2: Endereço
    ___:CEP{00000-000}
    row-gap4:
      @@+[Voltar](Step1)
      @[Próximo](Step3)

screen Step3:
  container-p8:
    # Passo 3: Confirmação
    > Revise seus dados
    row-gap4:
      @@+[Voltar](Step2)
      @[Finalizar](Success)

screen Success:
  container:
    # ✓ Sucesso!
    > Cadastro concluído
    @[Ir ao Dashboard](Dashboard)
```

</DslExample>

## Casos de Uso

### App com Tabs

<DslExample title="Navegação por Tabs" heightClassName="h-56">

```dsl
screen Feed:
  container:
    # Feed
    list:
      - Post 1
      - Post 2
  
  navigator:
    - Feed | Feed
    - Explorar | Explore
    - Notificações | Notifications
    - Perfil | Profile
```

</DslExample>

### Header com Navegação

<DslExample title="Header + Navegação" heightClassName="h-48">

```dsl
screen App:
  header:
    # Minha App
    row-end:
      @ghost[Início](Home)
      @ghost[Docs](Docs)
      @ghost[Perfil](Profile)
  
  container:
    # Conteúdo
```

</DslExample>
