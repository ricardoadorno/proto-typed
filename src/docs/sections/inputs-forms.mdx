
# Formulários e Inputs

import { DslExample } from "../../components/docs";

**Sintaxe principal:** `___: Label{placeholder}`

**Com tipo:** `___<tipo>: Label{placeholder}`

---

## Input de Texto

<DslExample title="Input Básico" description="Campo de texto simples">

```dsl
screen FormBasic:
  container:
    stack:
      ___: Nome{Digite seu nome}
      ___: Email{seu@email.com}
      ___: Telefone{(00) 00000-0000}
```

</DslExample>

## Tipos de Input

**Sintaxe:** `___<tipo>:` (opcional, padrão é text)

| Tipo     | Sintaxe           | Descrição           |
|----------|-------------------|---------------------|
| Text     | `___:`            | Texto simples       |
| Email    | `___email:`       | Email com validação |
| Password | `___password:`    | Texto mascarado     |
| Date     | `___date:`        | Seletor de data     |
| Number   | `___number:`      | Apenas números      |
| Textarea | `___textarea:`    | Texto multi-linha   |

<DslExample title="Tipos de Input" description="Inputs especializados">

```dsl
screen InputTypes:
  card:
    # Formulário
    ___: Nome{Digite seu nome}
    ___email: Email{seu@email.com}
    ___password: Senha{Digite sua senha}
    ___date: Data de Nascimento{Selecione}
    ___number: Idade{0}
    ___textarea: Mensagem{Digite sua mensagem}
```

</DslExample>

## Atributos Adicionais

**Sintaxe:** `| atributos` após o input

<DslExample title="Atributos" description="Required, placeholder, etc">

```dsl
screen FormAttributes:
  card:
    # Cadastro
    ___email: Email{seu@email.com} | required
    ___password: Senha{Mínimo 8 caracteres} | required minlength="8"
    ___: CPF{000.000.000-00} | required pattern="[0-9]{3}\.[0-9]{3}\.[0-9]{3}-[0-9]{2}"
```

</DslExample>

## Select (Dropdown)

**Sintaxe:** `___: Label{placeholder}[Opção1 | Opção2 | Opção3]`

<DslExample title="Select" description="Menu com opções">

```dsl
screen AddressForm:
  container:
    stack:
      ___: Estado{Selecione}[SP | RJ | MG | RS | PR]
      ___: País{Selecione}[Brasil | Argentina | Chile]
```

</DslExample>

## Checkboxes

**Sintaxe:**
- `[X] Label` → Marcado
- `[ ] Label` → Desmarcado

<DslExample title="Checkboxes" description="Seleção múltipla">

```dsl
screen Preferences:
  card:
    # Notificações
    [X] Receber emails
    [X] Notificações push
    [ ] SMS urgentes
    [ ] Newsletter
```

</DslExample>

---

## Radio Buttons

**Sintaxe:**
- `(X) Label` → Selecionado
- `( ) Label` → Não selecionado

<DslExample title="Radio Buttons" description="Seleção única">

```dsl
screen Survey:
  card:
    # Avaliação
    (X) Excelente
    ( ) Bom
    ( ) Regular
    ( ) Ruim
```

</DslExample>

## Formulário Completo

<DslExample title="Exemplo Completo" description="Todos os tipos de input">

```dsl
screen Registration:
  container:
    card:
      # Criar Conta
      ___: Nome Completo{João Silva}
      ___email: Email{joao@exemplo.com}
      ___password: Senha{Mínimo 8 caracteres}
      ___password: Confirmar Senha{Digite novamente}
      ---
      ___: País{Selecione}[Brasil | Portugal | Outro]
      ___: Estado{Selecione}[SP | RJ | MG]
      ---
      > Tipo de conta:
      (X) Pessoa Física
      ( ) Pessoa Jurídica
      > Notificações:
      [X] Aceito emails
      [X] Aceito termos
      row-between:
        @[Criar Conta](register)
        @outline[Cancelar](cancel)
```

</DslExample>

## Casos de Uso

### Login

<DslExample title="Formulário de Login" description="Email e senha">

```dsl
screen Login:
  container:
    card:
      # Bem-vindo
      ___email: Email{Digite seu email}
      ___password: Senha{Sua senha}
      [X] Lembrar de mim
      @[Entrar](login)
      >>> #[Esqueci minha senha](recover)
```

</DslExample>

### Busca com Filtros

<DslExample title="Busca e Filtros" description="Inputs e selects combinados">

```dsl
screen Search:
  container:
    stack:
      ___: Buscar{Nome do produto}
      ___: Categoria{Todas}[Eletrônicos | Roupas | Livros]
      ___: Ordenar{Relevância}[Preço | Avaliação | Vendas]
      [X] Frete Grátis
      [ ] Em Promoção
      @[Buscar](search)
```

</DslExample>

### Formulário em Grid

<DslExample title="Layout em Colunas" description="Grid para formulários complexos">

```dsl
screen ProductForm:
  container:
    grid-2:
      ___: Nome{Camiseta Básica}
      ___: SKU{ABC-123}
      ___number: Preço{0}
      ___number: Estoque{0}
      ___: Categoria{Selecione}[Roupas | Acessórios]
      ___: Marca{Selecione}[Nike | Adidas]
    ___textarea: Descrição{Descrição do produto}
    [X] Produto ativo
    @[Salvar](save)
```

</DslExample>
