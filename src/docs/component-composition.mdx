# Component Composition

## O que é
Padrão para criar componentes complexos a partir de componentes menores, reutilizando seções e repassando props através de múltiplos níveis.

## Capacidades
- Encadear componentes (`component HeaderBar` dentro de `component ProjectSection`).
- Reaproveitar layouts e props em diferentes contextos.
- Organizar código em “pedaços” lógicos para manutenção facilitada.

## Sintaxe
- `component HeaderBar:`
- `header:`
- `# %title`
- `row-end:`
- `@ghost[i-Search Buscar](Search)`
- `component ProjectSection:`
- `container:`
- `$HeaderBar:`
- `- Projetos`
- `grid-3 $ProjectCard:`
- `- ...`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| Componentes utilitários | Reuso local | Encapsulam partes repetidas (ex.: `HeaderBar`). |
| Componentes principais | Seção completa | Combinam utilitários e layouts. |
| `%props` compartilhados | Dados reutilizados | Permitem repassar valores para subcomponentes. |

## Casos de uso
- Criar seções compostas (header + grid) reaproveitadas em diferentes telas.
- Dividir componentes extensos em blocos utilitários (`HeaderBar`, `ActionCard`).
- Montar dashboards e catálogos com estrutura consistente.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
component HeaderBar:
  header:
    # %title
    row-end:
      @ghost[i-Search Buscar](Search)
      @ghost[i-Settings Config](Settings)

component ProjectCard:
  card:
    ## %name
    >>> %owner
    > %description
    @[Abrir](%destination)

component ProjectSection:
  container:
    $HeaderBar:
      - Projetos ativos
    grid-3 $ProjectCard:
      - Lançamento app mobile | Ana Souza | Status: Em QA | ProjectDetails
      - Campanha Q4 | Bruno Lima | Planejamento finalizando | CampaignPlan
      - Pesquisa usuários | Carla Dias | Entrevistas marcadas | ResearchBoard

screen Dashboard:
  $ProjectSection
```
</DslPreview>

<DslPreview title="Exemplo 2">
```dsl
component ActionCard:
  card:
    ## %title
    > %description
    row-end:
      @secondary[Detalhes](%destination)

component HomeSection:
  container:
    # %sectionTitle
    stack:
      $ActionCard:
        - Iniciar tour | Conheça o produto em minutos | Tour
      $ActionCard:
        - Ver exemplos | Aprenda com protótipos prontos | Examples

screen Home:
  container:
    $HomeSection:
      - Comece agora
```
</DslPreview>
