# Navigator

## O que é
Menu fixo (geralmente inferior em mobile) que permite alternar rapidamente entre telas relacionadas.

## Capacidades
- Declarar barra de navegação com `navigator:` seguido de itens `- Texto | Destino` (com ou sem ícones).
- Suportar ícones Lucide (`i-Home`, `i-User`) ao lado do texto.
- Integrar com qualquer `screen` para manter navegação consistente entre seções.

## Sintaxe
- `navigator:`
- `- Home | Home`
- `- i-List Tarefas | Tasks`
- `- Config | Settings`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `navigator:` | Cria barra | Repetir em cada tela do grupo para consistência. |
| `- Texto | Destino` | Item de navegação | `Destino` deve ser tela existente. |
| `- i-Icon Texto | Destino` | Item com ícone | Usar nomes Lucide válidos. |

## Casos de uso
- `navigator:` em telas principais de um app mobile (Home, Tarefas, Configurações).
- `navigator:` combinando ícones para facilitar reconhecimento rápido.
- Adicionar navegação inferior a dashboards com múltiplas subseções.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
screen Home:
  container:
    # Início
    > Explore as métricas e tarefas recentes.

  navigator:
    - i-Home Início | Home
    - i-List Tarefas | Tasks
    - i-Settings Config | Settings

screen Tasks:
  container:
    # Tarefas
    list:
      - Revisar backlog
      - Preparar release

  navigator:
    - i-Home Início | Home
    - i-List Tarefas | Tasks
    - i-Settings Config | Settings
```
</DslPreview>
