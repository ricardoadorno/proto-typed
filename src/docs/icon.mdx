# Icon

## O que é
Tokens de ícones (`i-IconName`) usados para reforçar ações e significados visuais em botões, listas e layouts Proto-Typed.

## Capacidades
- Inserir um ícone ao lado de textos em botões, listas e headings (`@[i-Download Exportar](...)`).
- Combinar com labels inline em `list`, `navigator`, `fab` ou `row`.
- Utilizar qualquer ícone da biblioteca Lucide referenciando o nome em PascalCase (`i-Download`, `i-Settings`).
- Misturar múltiplos ícones e texto na mesma linha através de tipografia padrão.

## Sintaxe
- `@[i-Download Exportar CSV](https://...)`
- `- i-Home Dashboard | Dashboard`
- `> i-Check Concluído`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `i-IconName` | Icone inline | Deve existir na biblioteca Lucide (PascalCase). |
| Uso com botões | Prefixo visual | Adicione antes do label dentro de `[ ]`. |
| Uso em listas/navigator | Indicador de item | Posicione antes do texto do item. |
| Tipografia | Decoração | Puede aparecer em títulos, parágrafos e notas. |

## Casos de uso
- `@[i-Plus Nova tarefa](TaskForm)` — Comunicação visual de “adicionar”.
- `list:` com `i-Home`, `i-Settings` — Menus e atalhos com ícones consistentes.
- `> i-Info` — Notas informativas dentro de cards.
- `fab:` — Complementar ações flutuantes com ícones claros.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
screen Shortcuts:
  container:
    # Atalhos rápidos
    list:
      - i-Home Dashboard | Dashboard
      - i-Calendar Agenda | Calendar
      - i-Settings Preferências | Settings

    row-start:
      @[i-Plus Nova tarefa](TaskForm)
      @outline[i-Download Exportar CSV](https://example.com/report.csv)
```
</DslPreview>

<DslPreview title="Exemplo 2">
```dsl
screen Status:
  container:
    # Status da implantação
    card:
      *> i-Info A implantação está em andamento.
      stack-tight:
        > i-Check Build concluída
        > i-Check Testes rodando
        > i-Clock Monitorando métricas

    navigator:
      - i-Home Início | Home
      - i-Server Deploys | Deploys
      - i-AlertCircle Logs | Logs
```
</DslPreview>
