# Container

## O que é
Layout base que delimita largura e padding do conteúdo, funcionando como “coluna” principal das telas Proto-Typed.

## Capacidades
- Oferecer envelopes com larguras distintas: `container`, `container-narrow`, `container-wide`, `container-full`.
- Centralizar automaticamente o conteúdo (`mx-auto`) e aplicar espaçamentos horizontais seguros.
- Conter qualquer outro layout ou elemento (stacks, rows, cards, formulários, botões).
- Facilitar composição com headers, footers e grids, mantendo consistência responsiva.

## Sintaxe
container:
  # Seção padrão
  > Texto de boas-vindas

container-narrow:
  card:
    ## Formulário focado

container-wide:
  grid-3:
    card:
      ## Métrica 1

container-full:
  row-end:
    @[Ação rápida](Action)
    @secondary[Suporte](Support)

## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `container` | Largura padrão | Ideal para textos, cards e listas gerais. |
| `container-narrow` | Coluna estreita | Use para formulários e cópia longa com foco. |
| `container-wide` | Coluna larga | Apropriada para dashboards e grids extensos. |
| `container-full` | 100% da largura | Depende do layout interno para definir padding. |
| Conteúdo interno | Slot principal | Pode receber `stack`, `row`, `card` e componentes custom. |

## Casos de uso
- `container: Página base com título, texto e CTA.`\n\
  `container:` `  # Bem-vindo` `  > Use o menu ao lado.`
- `container-narrow: Assistir a formulários ou leitura focada.`\n\
  `container-narrow:` `  card:` `    ## Cadastro`
- `container-wide: Dashboards com várias colunas.`\n\
  `container-wide:` `  grid-3:` `    card:` `      ## Métrica`

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
screen Overview:
  container:
    # Visão Geral
    > Use os cards abaixo para acompanhar o progresso semanal.

    stack-loose:
      card:
        ## Próxima reunião
        >> Segunda-feira, 10h
        @[Ver agenda](Meetings)

      card:
        ## Relatório completo
        > Faça download do consolidado mensal.
        @outline[i-Download Exportar PDF](https://example.com/report.pdf)
```
</DslPreview>

<DslPreview title="Exemplo 2">
```dsl
screen Dashboard:

      card:
        ## Bugs críticos

    card:
      ## Compartilhar relatório
      ___email: E-mail{usuario@empresa.com}
      row-end:
        @ghost[Cancelar](-1)
        @success[Enviar convite](sendInvite)
```
