# Screen

## O que é
Elemento raiz que define uma tela navegável. Cada `screen Nome:` cria um destino acessível por botões e links internos.

## Capacidades
- Declarar múltiplas telas no mesmo arquivo DSL (`screen Home`, `screen Dashboard`).
- Encapsular qualquer combinação de layouts (`container`, `stack`, `grid`) e componentes.
- Trabalhar com navegação via botões (`@[Ir para Dashboard](Dashboard)`) ou links (`#[Voltar](Home)`).
- Carregar modais/drawers complementares referenciando seus nomes.

## Sintaxe
- `screen NomeDaTela:`
- `container:`
- `# Cabeçalho`
- `> Conteúdo`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `screen Nome:` | Define uma view | Nome inicia com letra maiúscula. |
| Corpo da tela | Slot | Recebe layouts, componentes e exemplos DSL. |

## Casos de uso
- `screen Home:` Tela inicial com lista de atalhos e CTA principal.
- `screen Dashboard:` Visão analítica com `grid` e cards.
- `screen Settings:` Preferências com formulários e navegação auxiliar.
- `screen` + `modal/drawer:` Fluxos multi-etapas acionados por botões.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
screen Home:
  container:
    # Bem-vindo
    > Escolha uma ação para começar.

    row-between:
      @[Criar tarefa](TaskForm)
      @secondary[Ver dashboard](Dashboard)

    list:
      - Checklist diário | @[Abrir](DailyChecklist)
      - Exportar relatórios | @[Download CSV](https://example.com/report.csv)

screen TaskForm:
  container:
    # Nova tarefa
    ___: Título{Nome da tarefa}
    ___textarea: Descrição{Detalhes}
    row-end:
      @ghost[Cancelar](-1)
      @[Salvar](saveTask)

modal ConfirmDelete:
  card:
    # Confirmar exclusão
    > Esta ação não pode ser desfeita.
    row-end:
      @ghost[Voltar](close)
      @destructive[Excluir agora](deleteTask)
```
</DslPreview>

<DslPreview title="Exemplo 2">
```dsl
screen Dashboard:
  container-wide:
    # Visão geral
    grid-3:
      card:
        ## Leads
        # 1.240
      card:
        ## Conversões
        # 134
      card:
        ## Satisfação
        # 92%

  navigator:
    - Home | Home
    - Dashboard | Dashboard
    - Configurações | Settings

screen Settings:
  container-narrow:
    # Configurações
    ___email: E-mail{usuario@empresa.com}
    ___password: Senha{••••••••}
    row-end:
      @ghost[Cancelar](-1)
      @primary[Salvar alterações](saveSettings)
```
</DslPreview>
