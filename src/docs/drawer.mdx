# Drawer

## O que é
Painel lateral que desliza sobre a interface para exibir menus, filtros ou detalhes sem abandonar a tela atual.

## Capacidades
- Declarar `drawer Nome:` acionado por botões/links (`@[Abrir filtros](Nome)`).
- Conter listas, formulários e botões de confirmação/cancelamento.
- Integrar com `list` para criar menus ou `stack` para agrupar controles.
- Fechar com `@ghost[Fechar](close)` ou navegando para outra tela.

## Sintaxe
- `drawer FilterDrawer:`
- `container:`
- `# Filtros`
- `___select: Status{Todos | Ativo | Inativo}`
- `@ghost[Fechar](close)`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `drawer Nome:` | Define painel lateral | Nome em PascalCase. |
| Conteúdo interno | Slot | Normalmente `container` ou `stack`. |

## Casos de uso
- Menu lateral com listagem de telas (`drawer MainMenu`).
- Painel de filtros complementando um dashboard.
- Exibir detalhes de um item escolhido sem sair da tela principal.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
drawer MainMenu:
  container:
    # Menu
    list:
      - Dashboard | Dashboard
      - Tarefas | Tasks
      - Configurações | Settings

screen Tasks:
  header:
    # Minhas tarefas
    @ghost[i-Menu Menu](MainMenu)

  container:
    list:
      - Revisar backlog
      - Preparar release
```
</DslPreview>

<DslPreview title="Exemplo 2">
```dsl
drawer FilterDrawer:
  container:
    # Filtros avançados
    ___select: Status{Todos | Em andamento | Concluído}
    ___select: Responsável{Ana | Bruno | Carla}
    row-end:
      @ghost[Cancelar](close)
      @primary[Aplicar filtros](applyFilters)

screen Dashboard:
  container-wide:
    # Visão geral
    @[Abrir filtros](FilterDrawer)
    grid-3:
      card:
        ## Leads captados
        # 1.240
      card:
        ## Conversões
        # 134
      card:
        ## Net Promoter Score
        # 72
```
</DslPreview>
