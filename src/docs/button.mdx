# Button

## O que é
Botões acionam navegação, comandos e interações principais dentro das telas criadas com Proto-Typed.

## Capacidades
- Declarar botões com variantes (`primary`, `secondary`, `outline`, `ghost`, `destructive`, `link`, `success`, `warning`).
- Ajustar tamanhos (`-xs`, `-sm`, `-md`, `-lg`) para adequar altura e espaçamento.
- Encaminhar ações para telas, modais, drawers ou URLs externas (`@[Label](Destino)`).
- Atribuir ícones embutidos (`i-Download`) antes do texto para reforçar o contexto.
- Reagir a fluxos de confirmação (`(-1)` para voltar, chamar `modal`, `drawer` ou executar `alert('...')`).

## Sintaxe
- `@[Criar tarefa](TaskForm)`
- `@secondary-sm[Ver relatórios](Reports)`
- `@outline[i-Download Exportar CSV](https://example.com/report.csv)`
- `@ghost[Voltar](-1)`


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `@` / `@primary` | Ação principal | Variante padrão e mais evidente. |
| `@secondary`, `@outline`, `@ghost` | Ações complementares | Escolha conforme hierarquia visual. |
| `@destructive` | Operações críticas | Usar para deletar/remover. |
| `@link`, `@success`, `@warning` | Contextuais | Link abre destinos, success/warning reforçam status. |
| `-xs / -sm / -md / -lg` | Tamanho | Define altura/padding do botão. |
| `[Texto]` | Label | Obrigatório; pode incluir ícone `i-Icon`. |
| `(Destino)` | Ação | Tela (`Dashboard`), overlay (`ConfirmDelete`), link externo ou `-1` para voltar. |

## Casos de uso
- `@[Criar tarefa](TaskForm)` — CTA principal para abrir uma tela de formulário.
- `@secondary-sm[Ver histórico](History)` — Ação complementar em um card de métricas.
- `@outline[i-Download Exportar CSV](https://example.com/report.csv)` — Download de relatórios.
- `@ghost[Cancelar](-1)` — Retorna à tela anterior dentro de um fluxo multi-etapas.
- `@destructive[Excluir](ConfirmDelete)` — Dispara modal de confirmação destrutiva.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
screen Home:
  container:
    # Bem-vindo ao Proto-Typed
    > Gerencie tarefas e relatórios em um só lugar.

    row-between:
      @[Criar Tarefa](TaskForm)
      @secondary-sm[Ver Dashboard](Dashboard)

    card:
      ## Relatório semanal
      >> Exportar dados atualizados para o time financeiro.
      row:
        @outline[i-Download Exportar CSV](https://example.com/report.csv)
        @ghost[Compartilhar](ShareModal)

modal ShareModal:
  card:
    # Compartilhar link
    ___: Email{usuario@empresa.com}
    row-end:
      @ghost[Cancelar](close)
      @success[Enviar convite](sendInvite)

screen TaskForm:
  container:
    # Nova tarefa
    ___: Título{Nome da tarefa}
    ___textarea: Descrição{Detalhes}
    row-end:
      @ghost[Cancelar](-1)
      @[Salvar](saveTask)
```
</DslPreview>
