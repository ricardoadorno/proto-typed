# Link

## O que é
Elemento de navegação inline que cria hyperlinks para telas internas, modais, drawers ou URLs externas usando `#[Texto](Destino)`.

## Capacidades
- Navegar para outras telas Proto-Typed (`#[Ir para Dashboard](Dashboard)`).
- Abrir modais ou drawers existentes (`#[Abrir filtros](FilterDrawer)`).
- Disparar links externos, e-mail ou telefone (`https://…`, `mailto:…`, `tel:`).
- Ser utilizado dentro de parágrafos, listas e tabelas sem quebrar o fluxo textual.

## Sintaxe válida
- `#[Internal](ScreenName)`
- `#[Ajuda](HelpCenter)`
- `#[Visitar site](https://proto-typed.dev)`
- `#[Contato](mailto:time@empresa.com)`

## Sintaxe inválida
- `@[Label](Screen)` → usa `@`, não `#`.
- `#[Texto sem destino]` → precisa de `(Destino)`.
- `#[Texto](screen com espaço)` → nomes de tela não têm espaços.


## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `#[Texto](Screen)` | Navegação interna | Destino deve coincidir com nome de `screen`, `modal` ou `drawer`. |
| `#[Texto](https://...)` | Link externo | Abre a URL no navegador. |
| `#[Texto](mailto:...)` / `tel:` | Contato direto | Usa protocolo de e-mail/telefone. |
| Conteúdo inline | Slot | Pode envolver ícones (`i-`) e ênfases (`**`). |

## Casos de uso
- `#[Ver detalhes](DetailScreen)` — Link contextual dentro de cards ou listas.
- `#[Suporte](mailto:help@empresa.com)` — Canal rápido de ajuda.
- `#[Documentação](https://proto-typed.dev/docs)` — Referenciar material externo.
- `#[Voltar para o início](Home)` — Fornecer caminhos de navegação secundária.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
screen HelpCenter:
  container:
    # Central de ajuda
    > Consulte a #[Documentação oficial](https://proto-typed.dev/docs) para um guia completo.

    stack:
      card:
        ## Problemas comuns?
        #[Abrir troubleshooting](Troubleshooting)
      card:
        ## Fale com a gente
        #[Enviar e-mail](mailto:suporte@proto-typed.dev)

screen Troubleshooting:
  container:
    # Resolução de problemas
    > #[Voltar ao help center](HelpCenter)

    card:
      ## Erro de autenticação
      >> Revise suas credenciais ou #[Redefina a senha](RecoverPassword)
```
</DslPreview>
