# Card

## O que é
Contêiner com bordas e padding que agrupa conteúdo relacionado, ideal para destacar informações em blocos autônomos.

## Capacidades
- Criar cartões padrão (`card`), compactos (`card-compact`) e destacados (`card-feature`).
- Empilhar tipografia, botões, listas e qualquer elemento do DSL mantendo espaçamento interno.
- Aninhar outros layouts (`stack`, `row`) sem perder a aparência consistente.
- Servir como componente base para `component` reutilizáveis.

## Sintaxe
card:
  ## Destaque
  > Texto principal
  @[Ação](Destino)

card-compact:
  > Item compacto

card-feature:
  ## Plano premium
  *> Benefício exclusivo
  @[Falar com vendas](Contact)

## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `card` | Padrão | Padding médio, borda simples. |
| `card-compact` | Compacto | Menos padding para listas densas. |
| `card-feature` | Destaque | Borda reforçada/gradiente para CTAs premium. |
| Conteúdo interno | Slot | Recebe qualquer elemento do DSL (texto, botões, listas). |

## Casos de uso
- `card:` Informações chave com título, texto e CTA.
- `card-compact:` Listagens densas ou múltiplos mini cards em uma grade.
- `card-feature:` Destacar plano premium, anúncio ou seções hero secundárias.
- `card + stack:` Organizar formulários curtos com botões alinhados.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
screen Pricing:
  container:
    # Planos disponíveis
    grid-3:
      card:
        ## Starter
        >> Para equipes pequenas.
        @[Começar grátis](Signup)

      card-compact:
        ## Adicionais
        list:
          - Integrar com Slack
          - Exportar CSV
          - Suporte prioritário

      card-feature:
        ## Enterprise
        *> Atendimento dedicado e SLA personalizado.
        @[Falar com vendas](Contact)
```
</DslPreview>

<DslPreview title="Exemplo 2">
```dsl
component MetricCard:
  card:
    ## %title
    # %value
    >> %description

screen Dashboard:
  container-wide:
    # Indicadores principais
    grid-3:
      $MetricCard:
        - Leads | 1.240 | Captação nas últimas 24h
      $MetricCard:
        - Conversões | 134 | Fechamentos semanais
      $MetricCard:
        - Satisfação | 92% | Pesquisa pós-release

    card:
      ## Próximas ações
      list:
        - Revisar backlog | Hoje às 17h
        - Preparar release | Amanhã às 10h
      row-end:
        @ghost[Cancelar](close)
        @[Adicionar tarefa](TaskForm)
```
</DslPreview>
