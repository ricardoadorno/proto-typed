# Row

## O que é
Layout horizontal que posiciona elementos lado a lado, permitindo alinhar conteúdo, ações e métricas na mesma faixa.

## Capacidades
- Distribuir itens com alinhamentos diferentes: `row`, `row-start`, `row-center`, `row-between`, `row-end`.
- Combinar com `stack` internas para criar seções com título + botões no mesmo eixo.
- Suportar itens de tamanhos variados (cards, botões, inputs) mantendo espaçamento uniforme.
- Integrar CTAs com layout responsivo dentro de `card`, `container` ou `screen`.

## Sintaxe
row:
  @[Criar tarefa](TaskForm)
  @[Ver relatórios](Reports)

row-between:
  stack:
    > Plano Pro
    >>> Renovação em 30/09
  @[Alterar plano](ChangePlan)

row-center:
  @[Solicitar ajuda](Support)
  @[Abrir guia](Guides)

row-end:
  @ghost[Cancelar](-1)
  @[Salvar](saveTask)

## Variantes & Slots
| Variante/Slot | Papel | Notas |
| --- | --- | --- |
| `row` | Alinhamento padrão (start) | Usa gap consistente entre itens. |
| `row-start` | Força alinhamento à esquerda | Útil em headers com múltiplos botões. |
| `row-center` | Centraliza conteúdo | Ideal para estados vazios ou loaders horizontais. |
| `row-between` | Espaça extremos | Perfeito para título + ação lateral. |
| `row-end` | Alinha à direita | Facilita footers de cards com botões. |
| Conteúdo interno | Slot | Aceita qualquer elemento (botões, cards, textos, inputs). |

## Casos de uso
- `row-between:` Criar cabeçalhos com título à esquerda e botão à direita.
- `row-center:` Posicionar indicadores ou loaders no centro da tela.
- `row:` Exibir múltiplos cards com informações relacionadas lado a lado.
- `row-end:` Agrupar botões de confirmação/cancelamento no rodapé de um modal.

## Exemplos
<DslPreview title="Exemplo 1">
```dsl
screen Subscription:
  container:
    # Assinatura atual

    row-between:
      stack:
        ## Plano Pro
        >>> Renovação em 30/09
      @[Alterar plano](ChangePlan)

    row:
      card:
        ## Usuários ativos
        # 1.204
      card:
        ## Espaço utilizado
        # 78%

    row-end:
      @ghost[Cancelar](-1)
      @primary[Atualizar faturamento](BillingForm)
```
</DslPreview>

<DslPreview title="Exemplo 2">

```dsl
screen EmptyState:

        @outline[Importar CSV](ImportModal)

modal ImportModal:

    # Importar CSV
    ___file: Selecione um arquivo{}
    row-end:
      @ghost[Cancelar](close)

```
